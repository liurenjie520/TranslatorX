To navigate to an object in the Database tool window, press CtrlCtrl (Windows and Linux) or CommandCommand (macOS) and click the object.

Instead of navigating to the Database tool window, you can open a DDL definition of the object. To set this option as default, open settings &shortcut:ShowSettings;, navigate to Database | General, and clear the Prefer database view over DDL editor checkbox.

Also, you can always open a DDL definition of an object when you double-click it in the Database tool window. Open settings &shortcut:ShowSettings;, navigate to Database | General, and clear the Prefer data editor over DDL editor checkbox.
