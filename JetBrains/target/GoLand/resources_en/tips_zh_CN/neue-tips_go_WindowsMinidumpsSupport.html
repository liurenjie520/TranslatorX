Windows minidump 文件记录程序运行时的状态，或者在崩溃时记录。要读取 minidump 文件，您必须具有应用程序的二进制文件和转储文件。

您可以在 &productName; 中打开和浏览 minidump。要打开 minidump，请导航至 Run | Open Core Dump。指定可执行文件和转储文件的路径。点击 OK。

注意：只能在 Microsoft Windows 操作系统上创建 Windows minidump。但您可以在所有支持的操作系统上分析和调试应用程序。
