db_tip_consoles_scratch_files_and_buffers(控制台临时文件和缓冲区) [汉化反馈]

数据库控制台是已附加到数据源的 SQL 文件。创建数据源时，会自动创建数据库控制台。要打开控制台，右键单击数据源并选择 open console 或按 &shortcut:EditSource;。

临时文件是未附加到数据源的 SQL 文件。请将临时文件作为临时说明或草稿，用于在项目上下文之外的某些代码构思中。要创建临时文件，请导航到 file | New | scratch file(或按 &shortcut:NewScratchFile;)。

临时缓冲区是简单的文本文件，没有任何编码辅助功能。临时缓冲区可用于简单的任务列表和自己的注释。没有用于创建新临时缓冲区的操作的专用菜单项，但你可以使用 Find action 弹出窗口(&shortcut:GotoAction;)并运行 new scratch buffer 操作。
