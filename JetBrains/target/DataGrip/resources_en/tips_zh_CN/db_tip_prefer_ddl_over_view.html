db_tip_prefer_ddl_over_view(偏好 DDL 视图) [汉化反馈]

要导航到 Database 工具窗口中的对象，请按 CtrlCtrl(Windows 和 Linux)或 CommandCommand(macOS)并单击该对象。

你可以打开对象的 DDL 定义，而不是导航到 Database 工具窗口。若要将此选项设置为默认值，请打开设置&shortcut:ShowSettings;，导航到 Database | General，然后清除 Prefer database view over DDL editor 复选框。

此外，在 Database 工具窗口中双击对象时，始终可以打开该对象的 DDL 定义。打开设置 &shortcut:ShowSettings;，导航到 Database | General，然后清除 Prefer data editor over DDL editor 复选框。
